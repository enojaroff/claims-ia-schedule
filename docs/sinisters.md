# Types de sinistres

Le type de sinistre est représenté par un code alphanumérique, qui a une des valeurs suivantes :

Code		| Sinistre
--------|---------
DDE			|	Dégât des eaux
DE			|	Dommages électriques
VOL			|	Vol
INC			|	Incendie
CATNAT	|	Catastrophe naturelle
BDG			|	Bris de glace
CVT			|	Choc de véhicule terrestre
SEC			|	Sécheresse
